# 📸 Responsive Photo Gallery Website

A modern and fully responsive photo gallery website built using HTML, CSS, and JavaScript.  
This project showcases beautiful images with category filtering, smooth animations, and an interactive user interface.

---

## 🚀 Features

- Responsive Design (Mobile Friendly)
- Dynamic Image Filtering (Nature, City, People, Animals)
- CSS Grid & Flexbox Layout
- Smooth Scrolling Navigation
- Animated Hover Effects
- Mobile Hamburger Menu
- Newsletter Subscription UI
- Blog Section Layout

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (Vanilla JS)
- Font Awesome Icons

---


---

## 🎯 How It Works

- Gallery images are dynamically loaded using JavaScript.
- Users can filter images by category.
- Navigation highlights active sections while scrolling.
- Fully responsive layout adapts to all screen sizes.

---

## 📸 Preview

This project demonstrates:
- CSS Grid for gallery layout
- Flexbox for navbar and categories
- JavaScript DOM manipulation
- Smooth UI animations
